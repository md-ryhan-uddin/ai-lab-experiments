## Experiments
Each experiment is designed to be completed during a laboratory session and follows the standard lab report format.

| No | Title | Key Topics |
|---|-------|------------|
| 01 | Introduction to Python, NumPy, Pandas, and Data Visualization for AI | Python basics, NumPy arrays, Pandas, Matplotlib, Iris dataset |
| 02 | State Space Search Algorithms | BFS, DFS, A* search, grid-maze navigation |
| 03 | Knowledge Representation and Rule-Based Expert Systems | Forward chaining, inference engine, explanation trace |
| 04 | Supervised Machine Learning Algorithms | Decision Tree, KNN, Naive Bayes, SVM, ROC-AUC |
| 05 | Unsupervised Learning and Dimensionality Reduction | PCA, K-Means, Hierarchical Clustering, DBSCAN |
| 06 | Perceptron and Multilayer Perceptron (MLP) | Single-layer perceptron, MLP, early stopping |
| 07 | Feedforward Neural Networks with Backpropagation | Backpropagation, Adam optimizer, learning curves |
| 08 | Image Classification with CNNs and Transfer Learning | Convolutional layers, pretrained models, fine-tuning |
| 09 | NLP with RNNs and LSTMs | Sequence modeling, sentiment analysis, IMDB dataset |
| 10 | Text Generation with Transformers and LLMs | Hugging Face pipelines, tokenization, zero-shot classification |

## Run the Notebooks
Choose your preferred platform:

- 💻 **Jupyter Notebook** – Local execution
- ☁️ **Google Colab** – No installation required
- 🏆 **Kaggle Notebooks** – Free GPU (recommended for Experiments 08–10)

## Usage

Open any `experiment_XX.ipynb` in Jupyter and run the cells top to bottom.
Each notebook prints `Setup complete.` (or similar) once its dependencies
load correctly. Some later experiments (08–10) use TensorFlow/Keras or
Hugging Face Transformers, which are heavier installs — see the per-notebook
"Required Software and Libraries" section before running.
